## نحوه استفاده

۱. این مخزن را فورک کنید
۲. به بخش Actions بروید و روی I understand my workflows, go ahead and enable them کلیک کنید
۳. Settings -> Actions -> General -> Workflow permissions -> انتخاب Read and write permissions -> کلیک کنید Save
۴. با استفاده از این [راهنما](https://github.com/yt-dlp/yt-dlp/wiki/FAQ#how-do-i-pass-cookies-to-yt-dlp) کوکی دریافت کنید
۵. کوکی‌ها را به base64 کدگذاری کنید
۶. یک secret به نام `COOKIES` کدگذاری شده در base64 اضافه کنید در Settings -> Secrets and variables -> Actions -> New repository secret
۷. یک branch به نام `downloads` ایجاد کنید
۸. ورکفلو را با لینک ویدئو اجرا کنید

> [!NOTE]
> برای وب‌سایت‌های دیگر از ورکفلو بدون کوکی استفاده کنید
